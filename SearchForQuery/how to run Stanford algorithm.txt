To run Stanford algorithm, you have to run a java server first.
For that, you have to follow first 5 stepts from next web page:
https://www.maketecheasier.com/run-java-program-from-command-prompt/

After you set the path to the jdk, you have to enter in console next commmand:
java -Xmx4g -cp "*" edu.stanford.nlp.pipeline.StanfordCoreNLPServer -serverProperties StanfordCoreNLP-spanish.properties -port 9000 -timeout 15000
